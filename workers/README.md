# tako-mcp workers

[![workers-ci](https://github.com/TakoData/tako-mcp/actions/workflows/workers-ci.yml/badge.svg)](https://github.com/TakoData/tako-mcp/actions/workflows/workers-ci.yml)

This subdirectory hosts the TypeScript Cloudflare Workers port of the Tako MCP server, tracked under the [tako-mcp Linear project](https://linear.app/trytako/project/tako-mcp-635ac07fae22).

## Local dev

```sh
npm ci
npm test
npm run typecheck
npm run dev
```
